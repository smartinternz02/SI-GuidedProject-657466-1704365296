Problem:

Diabetic Retinopathy(DR) is a common complication of diabetes mellitus which causes lessions on the retina affecting the vision.If it is not detected early, it can lead to blindness.DR is not a reversible process and treatment only sustains the vision. DR early can reduce the risk of blindness.

Solution:

In this project, we will be building a Transfer model that can detect and classify types of Diabetic Retinopathy.A web application is integerated with the model, from where the user can upload a Diabetic Retinopathy image and can classify which type of DR it is and see the result on the User Interface
